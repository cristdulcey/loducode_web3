# Loducode web3

Basic components for the development of loducode s.a.s.

### install

`pip install loducode_web3==0.0.1`

## functionalities

- **Admin**
    - AttributeLAndAdmin
    - AttributeNftAdmin
    - LandAdmin
    - NftAdmin
- **Models**
    - AttributeLand
    - AttributeNft  
    - Land  
    - Nft
- **Urls Api**
    - /api/nfts/
    - /api/attributes_nft/
    - /api/lands/
    - /api/attributes_lands/
- **Views api**
    - AttributeLandViewSet
    - AttributeNftViewSet
    - LandViewSet
    - NftViewSet
